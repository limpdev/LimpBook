# Introduction

Zenity enables you to create the following types of simple dialog: 

  * Calendar

  * File selection

  * Forms

  * List

  * Notification icon

  * Message

    * Error

    * Information

    * Question

    * Warning

  * Password entry

  * Progress

  * Text entry

  * Text information

  * Scale

  * Color selection
